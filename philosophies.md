# Source material — show philosophy

This file is the canonical context any episode draws from. Update it when
your thinking shifts; the next script pass will pull from this version.

## What this is

**Land the Plane** is an experimental, AI-produced podcast about
software engineering, AI-assisted development, and engineering
leadership in the agentic era. Research, drafting, audio synthesis,
and publishing all run through an AI pipeline. The voice is synthetic
(Piper TTS). Episodes are working drafts of an argument, not
authoritative reporting.

## Show positioning
- Weekly podcast (~30 minutes) about software engineering, AI-assisted
  development, and what it actually takes to lead engineering teams in
  the agentic era.
- Solo essay-and-news format. Each episode pairs a current-events segment
  with one longer argument.
- Voice: opinionated, plain-spoken, allergic to corporate-generic
  language. Skeptical of unjustified hierarchy and management theater.
- 25+ years in the industry; 10+ years in engineering management. The
  host's experience shows up as perspective, never as autobiography or
  employer name-checking.

## Editorial stance
- Management philosophy is "anarchistic" — autonomy, psychological
  safety, distributed authority, authenticity.

## Core technical interests

**Platform engineering & GitOps**
- Kubernetes, GKE, Kind, Tilt
- ArgoCD, Tekton, Helm, raw YAML, CRDs, controllers
- Three-repo GitOps model (application repos, version/environment
  manifests, deployment repos)
- CI/CD pipeline architecture, developer experience

**AI-assisted development**
- Active Claude Code user
- Stream Deck (15-key) integration for monitoring Claude session status
  and triggering dev actions
- ZSA Moonlander Mark II split keyboard
- tmux for remote terminal access from phone
- "ralph" — bash script implementing a REPL-like agentic loop using
  Claude CLI

**Current big idea: the Intent Layer**
- Product vision: human conversations as the primary input to software
  creation, code as a downstream artifact
- "Intent Layer Captain's Log" — personal tool capturing intent, pivots,
  reasoning from Claude Code sessions alongside diffs
- Intended as both a recall tool and source material for a book on
  intent-driven development

## Management & leadership themes
- Psychological safety
- Distributed authority
- Authentic professional communication (built a custom Claude skill for
  emails reflecting personal values rather than corporate-generic
  language)

- Empathy and reflective practice

- Iterative refinement workflow
